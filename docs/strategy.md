# Strategy — media2rag

## Phase 1: Personal Tool (Now)

**Цель:** Личный AI-эксперт, который растёт вместе со мной.

- Загружаю книги, видео, заметки, исследования
- AI отвечает на основе моих данных, не обучения
- С каждым документом — экспертнее
- Помнит контекст между сессиями

**Метрика:** Я сам пользуюсь каждый день. 50+ документов, точные ответы.

---

## Phase 2: Real-World Test (Father's Window Business)

**Цель:** Проверить на реальном бизнесе.

Оконный бизнес — идеальный первый кейс:

| Данные | Что даёт |
|--------|----------|
| Каталог окон, профилей, фурнитуры | AI знает все продукты |
| Скрипты продаж | Анализ звонков менеджеров |
| Регламенты монтажа | Ответы на вопросы бригад |
| FAQ клиентов | Быстрые ответы менеджерам |
| Обучение | Новый сотрудник → AI-наставник |

**Преимущества:**
- Доступ к реальным данным
- Доступ к реальным пользователям (менеджеры, монтажники)
- Сеть предпринимателей через отца
- Можно измерить ROI (конверсия, время обучения, ошибки)

**Метрика:** Бизнес использует, есть измеримый эффект.

---

## Phase 3: Market (If Phase 2 Works)

**Цель:** Масштабирование.

Если оконный бизнес показал результат → другие ниши:
- Консалтинг
- Производство
- Обучение
- Юриспруденция

**Модель:** SaaS подписка за компанию.

## Why This Order

1. **Для себя** → делаешь качественно, решаешь реальную боль
2. **Через отца** → тест на реальных данных без маркетинга
3. **Рынок** → только если 1 и 2 работают

Не наоборот. Не "сначала продукт для рынка".
