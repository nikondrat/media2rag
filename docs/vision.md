# Vision — media2rag

## Strategy

1. **Phase 1 (сейчас):** Личный AI-эксперт — загружаю данные, AI отвечает на их основе, растёт со мной
2. **Phase 2:** Тест на оконном бизнесе отца — реальные данные, реальные пользователи, измеримый ROI
3. **Phase 3:** Масштабирование на другие ниши — только если Phase 2 работает

Подробнее: `docs/strategy.md`

---

## Current Focus: Core Engine

**Personal RAG system.** Загрузил свои данные (книги, видео, заметки) → AI отвечает на основе них, с источниками, помнит контекст между сессиями.

**Phase 1 — Core Engine (сейчас):**
- `process` — обработка любых данных в RAG-формат
- `chat` — вопросы по своим данным с источниками
- `memory` — память между сессиями

**Это приоритет. Всё остальное — потом.**

## Why Core Engine Matters

AI-агенты (ChatGPT, Claude) работают с **сырыми данными**. media2rag готовит данные:

```
Сырой текст → CTG Pipeline → Структурированный RAG-документ
```

Результат: ответы точнее, с источниками, на основе твоих данных, не обучения.
